# Contributing

## Working Rules

1. Open or reference an issue before changing behavior.
2. Keep changes scoped to the issue.
3. Run the relevant Arduino CLI build and verification steps.
4. For fixes and feature completion, request a subagent review before closing the issue.
5. Do not modify the upstream reference materials directory; copy only the needed files into this project.

## Close Criteria

- Code or docs landed on the default branch
- Verification evidence attached
- Subagent review completed
- Issue linked to the change set
